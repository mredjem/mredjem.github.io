## Description

This is my personal website built with Jekyll and hosted on GitHub Pages.

## Installation

  * Install Jekyll
  * Start the website locally

````
sudo gem install jekyll
````

Start the local server.

````
jekyll serve
````

Navigate to [http://127.0.0.1:4000/](http://127.0.0.1:4000/).
